Machine Learning-Based Fault Detection for Electrical Tomography

This industrial project focused on developing a machine learning model capable of identifying faulty measurements in electrical tomography data. The work involved data preprocessing, feature engineering, supervised learning and model evaluation using real industrial measurement data. The developed classification model achieved approximately 97% classification accuracy on the test dataset.

Project Overview

Electrical tomography systems generate large amounts of measurement data, where faulty measurements may reduce measurement quality and system reliability. The objective of this project was to develop an automated machine learning solution capable of identifying abnormal measurements.
