Respiratory Rate Estimation from Smartphone Accelerometer Data

This project focuses on developing an algorithm that estimates respiratory rate from three-axis accelerometer data recorded with a smartphone placed on the subject's chest. The estimated respiratory rate is validated against a reference respiratory belt (BIOPAC MP150).

Overview

Respiration causes subtle movements of the chest that can be detected by a smartphone accelerometer. However, these movements are often masked by body motion, sensor orientation changes and measurement noise.

The objective of this work is to develop a robust signal processing pipeline capable of extracting respiratory information from raw accelerometer measurements and estimating respiratory rate under different measurement conditions.

My Contribution

Signal Processing Pipeline

  1. Import raw accelerometer data.
  2. Interpolation of non-uniform sampling.
  3. Filtering and preprocessing.
  4. Principal Component Analysis (PCA).
  5. Respiratory signal extraction.
  6. Autocorrelation analysis.
  7. Peak detection.
  8. Respiratory cycle detection.
  9. Respiratory rate estimation.
  10. Confidence estimation.

Methods

MATLAB Signal Processing PCA Autocorrelation Digital Filtering Peak Detection Interpolation Statistical Analysis

Results

The developed algorithm successfully extracts respiratory information from chest-mounted smartphone accelerometer measurements and estimates respiratory rate by combining signal preprocessing, principal component analysis and autocorrelation-based cycle detection.

Performance is evaluated by comparing the estimated respiratory rate with reference measurements obtained using a BIOPAC respiratory belt.

Future Work

Future work includes improving robustness during motion, evaluating the algorithm on larger datasets, investigating machine learning approaches for respiratory signal extraction and implementing the method for real-time operation on mobile devices.
