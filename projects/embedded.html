Embedded Sensor Systems and Real-Time Signal Processing

This project collection focuses on developing embedded software for microcontroller-based sensor systems. The work includes real-time data acquisition, sensor communication, signal processing and orientation estimation using inertial measurement units (IMUs).

Several prototype systems were implemented using Arduino-based microcontrollers and a variety of sensors, providing practical experience in embedded software development and hardware integration.

Project Overview

Embedded systems frequently combine multiple sensors that must operate in real time with limited computational resources. These projects focused on developing efficient firmware for acquiring sensor data, processing measurements and estimating device orientation.

The implementations emphasized reliable sensor communication, efficient data processing and modular software architecture. Measurement data and calculated orientation information were displayed in real time using OLED-based user interfaces.

My Contribution

  • Developed embedded software using C++.
  • Implemented communication between microcontrollers and multiple sensors.
  • Acquired and processed accelerometer and gyroscope measurements.
  • Implemented filtering and real-time orientation estimation.
  • Developed sensor-fusion methods.
  • Designed OLED-based user interfaces.
  • Debugged embedded hardware and software.

System Workflow

  1. Initialize the microcontroller and connected sensors.
  2. Acquire accelerometer and gyroscope measurements.
  3. Filter and process the sensor data.
  4. Estimate the orientation of the device.
  5. Display the calculated results.
  6. Repeat the measurement cycle in real time.
