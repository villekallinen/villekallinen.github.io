Master's Thesis · Biomedical Signal Processing

Respiratory Rate Estimation from Smartphone Accelerometer Data

This Master's thesis investigates whether respiratory rate can be estimated accurately from chest-mounted smartphone accelerometer measurements. The project combines biomedical signal processing, statistical analysis and algorithm development to extract respiratory information from three-axis accelerometer data. The developed algorithm is currently under evaluation using experimental measurements and is being validated against a BIOPAC MP150 respiratory belt, which serves as the reference measurement.

Project Overview

Respiration causes subtle chest movements that can be measured using modern smartphone accelerometers. However, these signals are often masked by body motion, sensor orientation changes and measurement noise, making reliable respiratory monitoring a challenging signal processing problem.

The objective of this work is to develop a robust signal processing pipeline capable of extracting respiratory information from raw accelerometer measurements and estimating respiratory rate under realistic measurement conditions.

Signal Processing Pipeline

  1. Import raw accelerometer measurements.
  2. Interpolate non-uniformly sampled sensor data.
  3. Preprocess and filter the signals.
  4. Perform Principal Component Analysis (PCA).
  5. Extract the respiratory signal.
  6. Estimate respiratory periodicity using autocorrelation.
  7. Detect breathing peaks and respiratory cycles.
  8. Estimate respiratory rate.
  9. Compute confidence metrics.
