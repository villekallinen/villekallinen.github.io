Machine Learning-Based Fault Detection for Electrical Tomography

This industrial project focused on developing a machine learning model capable of identifying faulty measurements in electrical tomography data. The work involved data preprocessing, feature engineering, supervised learning and model evaluation using real industrial measurement data.

The developed classification model achieved approximately 97% accuracy on a held-out test dataset.

Project Overview

Electrical tomography systems generate large volumes of measurement data. Faulty or abnormal measurements can reduce data quality and affect the reliability of subsequent analysis. The objective of this project was to develop an automated machine learning solution capable of distinguishing faulty measurements from normal measurement conditions.

The project covered the complete supervised machine learning workflow, from preparing and analysing the raw measurement data to training and evaluating the final classification model.

Project Outcome

The developed Random Forest classifier achieved approximately 97% classification accuracy on the test dataset. The model successfully distinguished between normal and faulty measurement conditions, demonstrating the potential of machine learning for automated measurement-quality monitoring.
