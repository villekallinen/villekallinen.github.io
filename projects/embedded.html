Embedded Sensor Systems and Real-Time Signal Processing

This project collection focuses on developing embedded software for microcontroller-based sensor systems. The work includes real-time data acquisition, sensor communication, signal processing and orientation estimation using inertial measurement units (IMUs). Several prototype systems were implemented using Arduino-based microcontrollers and a variety of sensors to gain practical experience in embedded software development.

Project Overview

Embedded systems frequently combine multiple sensors that must operate in real time with limited computational resources. This project focused on developing efficient firmware for reading sensor data, processing measurements and estimating device orientation.

The implementation emphasized reliable communication, efficient data processing and modular software architecture.
